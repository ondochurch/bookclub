📊 북클럽 참여 현황

현재 진행 중인 북클럽의 참여자 명단과 출석 현황을 확인할 수 있습니다.

🔧 Airtable 설정 방법

1단계: Airtable 계정 만들기

1. airtable.com/signup 방문

2. 이메일로 무료 계정 생성 (Google 계정 연동 가능)

3. 계정 생성 완료 후 로그인

2단계: Base 만들기

1. "Start from scratch" 또는 "북클럽 참여자" 템플릿 선택

2. Base 이름: "온도교회 북클럽 참여자"

3. 필드 추가:

  • 이름 (Single line text)
  • 연락처 (Phone number)
  • 참여 회차 (Multiple select)
  • 출석 현황 (Checkbox)
  • 비고 (Long text)

3단계: 공유 링크 가져오기

1. Airtable Base 우측 상단 "Share" 클릭

2. "Create a shareable link" 토글 활성화

3. "Embed this base" 클릭

4. iframe 코드 복사

4단계: 웹사이트에 추가

1. participants.html 파일 열기

2. 79-86번째 줄의 주석 제거 (<!--와 --> 삭제)

3. YOUR_AIRTABLE_SHARE_LINK_HERE를 복사한 링크로 교체

4. 88-176번째 줄 (이 설정 안내) 삭제

5. 변경사항 저장 및 커밋

💡 팁: Airtable의 "Form" 기능을 사용하면 참여자들이 직접 정보를 입력할 수 있습니다. Forms → "Create form" → 링크 공유

🔒 개인정보 보호: 민감한 정보는 공개 공유 대신 "Restricted view"를 사용하세요.
